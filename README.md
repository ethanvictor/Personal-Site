# Personal Webpage

Static multi-page personal website scaffold for Ethan Victor.

## Included pages

- `index.html` - Home page
- `academic.html` - Academic achievements, modules, goals, research interests
- `professional.html` - CV, experience, skills, professional links
- `personal.html` - Hobbies, interests, and side projects
- `template.html` - Reusable starter for any future page

## How to add your own details

1. Search for text like `[YOUR_EMAIL_HERE]` or `[Add your content here.]`
2. Replace the placeholder text with your own details
3. Put images in `assets/images/`
4. Put PDFs or other files in `assets/files/`
5. Keep the filenames the same if you want the existing links to keep working automatically

## How to add a new page

1. Duplicate `template.html`
2. Rename it, for example `research.html`
3. Replace the placeholder title, page heading, and content
4. Add a link to the new page in the navigation on each page, or copy the navigation block from the updated template once you've added it there
